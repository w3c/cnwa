# Web 适老化及无障碍线上交流会・2021年5月29日

[W3C Web 无障碍中文社区组](https://www.w3.org/community/cnwa/) 将于**2021年5月29日**举行 Web 适老化及无障碍线上交流会，欢迎于**5月25日前**注册参会。同时欢迎大家加入[W3C Web 无障碍中文社区组](https://www.w3.org/community/cnwa/)参与社区组的活动。

随着老龄化程度不断加深，让老龄用户也能无障碍地使用 Web 将有效保障他们平等参与社会活动，从而共享 Web 资源及权益。本次交流会将围绕 Web 上的老龄用户展开，分享和探讨目前已知的老龄用户需求以及相关的国内、国际标准，专注于 Web 适老化及无障碍等技术话题。

本次会议主题将涵盖但不限于：
1. 探讨与年龄相关的 Web 使用障碍；
2. 如何改进 Web 设计以确保老年人的使用需求；
3. 现有的国家及国际无障碍标准如何满足老年人的需求；
4. 未来在该领域的潜在工作方向。

- 国际相关标准及国外厂商实施
  - **W3C Web 无障碍国际标准及适老化** Judy Brewer (W3C 无障碍标准计划总负责人，1997年开始致力于 Web 无障碍的标准工作，引导及参与了一系列影响深远的 Web 无障碍标准的制定，同时她也一直在致力于确保 Web 技术可以为残障人士使用；为网页内容、浏览器、媒体播放和编译工具建立操作指南；提升现有资源为网站进行评估和维护；为W3C开发Web信息无障碍化的教育和推广资源；并参与未来网络信息无障碍化的研发等工作，是 W3C Web 无障碍的奠基者之一。)
  - **国外厂商针对无障碍规范的实施** 丁尧 （Facebook，脸书无障碍研究主管，威斯康辛大学人机交互与康复-特殊教育博士。多年来从事残疾人、老年人的用户体验研究，帮助提升产品无障碍、消弭数字鸿沟，同时探索人工智能、虚拟现实等新技术在无障碍方面的应用。参与多项无障碍设计开发规范和标准工作，包括 WCAG, Section 508, EN 301 549, CVAA 等。）

- 国家标准及政策分享
  - **中国信息无障碍国家标准在适老化方面的相关技术要求** 王炜（浙江大学，致力于信息无障碍标准及测试等工作，信息无障碍国家标准起草者。）
  - **《移动互联网应用（APP）适老化通用设计规范》解读** 丁丽婷 （中国信息通信研究院产业与规划研究所工程师，从事信息无障碍、信息化、数字化转型等领域的研究工作，参与支撑工信部制定《关于推进信息无障碍的指导意见》、《互联网应用适老化及无障碍改造专项行动方案》等政策文件，《移动互联网应用（APP）适老化通用设计规范》起草人。）

- 国内厂商围绕无障碍的实施经验技术分享
  - **老年人使用移动应用面临的障碍及改进方向** 吴益明 （信息无障碍研究会，高级信息无障碍工程师，视障者，中国最早一批信息无障碍工程师之一，拥有专业的信息无障碍知识体系，负责Android应用无障碍解决方案研究工作，并主笔编写《Android产品无障碍开发指南》；曾为包括腾讯、阿里巴巴、百度、华为等在内的众多知名公司产品提供专业服务。）
  - **Web无障碍化解决方案与腾讯文档实践** 李鑫（腾讯 Alloyteam）
  - **用户参与式开发经验和理念普及工作** 朱浠汐（小米）
